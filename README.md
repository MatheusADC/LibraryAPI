# <sub><img src="https://img.icons8.com/?size=100&id=7XI6kQrF2s0K&format=png&color=000000" alt="Prompt icon" width="32"></sub> Commands
### Create project
```
npm init -y
```

### Install the dependencies
```
npm i
```

### Run server
```
node server.js
```

### Run NoSQL server
```
node serverNoSQL.js
```
